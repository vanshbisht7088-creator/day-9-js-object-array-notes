JavaScript Objects and Arrays
📌 Overview

This project demonstrates the basics of JavaScript Objects and Arrays. It covers creating objects, modifying their properties, looping through object data, and performing various array operations using built-in JavaScript methods.

The aim of this task is to understand how data can be stored and manipulated efficiently in JavaScript.

🎯 Objectives
Create and work with JavaScript objects.
Add, update, and delete object properties.
Iterate through object properties using loops.
Create an array of student objects.
Use array methods such as:
map()
filter()
find()
reduce()
sort()
📂 Topics Covered
1. JavaScript Objects

Created objects for:

Employee
Mobile
Car

Performed operations such as:

Adding new properties
Updating existing properties
Deleting properties
2. Object Iteration

Used the for...in loop to access all properties of an object.

3. Arrays of Objects

Created an array containing details of five students.

4. Array Methods
map()

Used to create a new array containing only student names.

filter()

Used to get students whose marks are greater than 80.

find()

Used to search for a particular student from the array.

reduce()

Used to calculate the total marks of all students.

sort()

Used to arrange students according to their marks in ascending and descending order.
